# Multiplayer Online Checkers Game

Multiplayer Checkers game made with PhaserJS, React, NestJS, SocketIO (TypeScript).

## Server
```bash
yarn install
yarn start:dev
```
### or
```bash
npm install
npm run start:dev
```

## Client
### Do not forget to create .env file based in .env.example file in `/client` folder path root
```bash
yarn install
yarn dev
```

### or
```bash
npm install
npm run dev
```

## Concurrently
```bash
npm run dev
```
### or
```bash
yarn dev
```
